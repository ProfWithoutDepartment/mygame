Проект: Game_Fedorov_Slova

Структура:
- index.html - страница авторизации
- levels.html - выбор уровня
- final.html - рейтинг игроков
- common.css, common.js - общие стили и скрипты
- level1/, level2/, level3/ - папки с уровнями игры

Как запустить:
1. Создайте папку Game_Fedorov_Slova
2. Поместите все файлы в соответствующие папки
3. Откройте index.html в браузере

Уровни:
1. Синонимы для картинки - выбираем правильные синонимы
2. Слова по цвету и признаку - выбираем слова по двум критериям
3. Чужеродные фрагменты - находим неподходящие слова в тексте

Все данные сохраняются в localStorage.
